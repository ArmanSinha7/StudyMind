📚 StudyMind AI

An all-in-one AI-powered study companion built for engineering students.
No installation. No backend. Just open and study.

✨ Features
- 🤖 AI Q&A Chatbot — ask anything, get instant subject-aware answers
- 📝 Quiz Generator — auto-generates MCQs on any topic with difficulty tags
- 🃏 Flashcards — pre-loaded decks with Easy / OK / Hard self-rating
- 📅 AI Study Planner — personalized day-by-day exam prep schedules
- 📄 Summarizer — paste notes, get bullet-point summaries instantly
- 🔢 Formula Sheet — searchable reference cards for Algorithms, Math, Physics & Networks
- ⏱️ Pomodoro Timer — focus sessions with circular countdown and session logging
- ✅ Habit Tracker — weekly grid with streaks and check-in stats
- ❌ Mistake Log — auto-logs wrong answers for targeted revision
- 🏆 Achievements — XP system, level progression, coins, and 12 unlockable badges

🛠️ Tech Stack
HTML5 · CSS3 · Vanilla JavaScript · Anthropic Claude API · localStorage

🚀 Getting Started
Just open index.html in any modern browser. No setup required.
Add your Anthropic API key in the settings to unlock AI features.
